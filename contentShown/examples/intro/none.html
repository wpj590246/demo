此处演示的是某视屏网站内容，
演示的参数时nConte即导航内容为content时和listPlace即导航位置居中(左、右、上、下 请使用样式表中的position控制) 的情况，
这种情况下，导航的样式完全由样式表控制，在显示内容规则的情况下你任然可以使用循环，背景滑动，导航按钮等设置

	"nContent"	: "none",
	"listPlace"	: "center",
	"navNum"	: 12,
	"navSpace"	: 27,
	"pWidth"	: 960,
	"pHeight"	: 340,
	"tbgAnimate" 	: false,
	"animate"	: "left",
	"tipsAnimate"	: "slide",
	"loop"		: false,
	"tipsBtn"	: true,
	"data"		: [......]
